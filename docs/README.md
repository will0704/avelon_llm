## 🤖 About avelon_llm

This repository contains the Python AI microservice for:
- Document classification (ResNet-18)
- OCR text extraction (Tesseract)
- Named Entity Recognition (BERT-based)
- Fraud detection
- Credit scoring (XGBoost)

See the global docs for detailed AI pipeline specifications.
